HOW TO EXTRACT FROM STOCK ROM
-------------------------------
1. Download your stock ROM zip for the Infinix X663.
2. Unzip the stock ROM zip file.
3. Use lpunpack to unpack super.img (if present):
       lpunpack super.img ./out/
4. Mount the output images to copy files:
       mkdir /tmp/vendor && mount -o loop out/vendor.img /tmp/vendor
       mkdir /tmp/system && mount -o loop out/system.img /tmp/system
5. Use magiskboot to extract kernel + dtb from boot.img:
       magiskboot unpack boot.img
   This produces: kernel  →  copy to prebuilt/kernel
                  dtb     →  copy to prebuilt/dtb.img

========================================================
  QUICK EXTRACTION COMMANDS SUMMARY
========================================================

# Step 1 — Extract kernel and dtb from boot.img
magiskboot unpack boot.img
cp kernel path/to/device/tree/prebuilt/kernel
cp dtb    path/to/device/tree/prebuilt/dtb.img

# Step 2 — Unpack super.img (if stock ROM uses it)
lpunpack super.img ./super_out/

# Step 3 — Mount vendor partition and copy blobs
mkdir -p /tmp/vendor
mount -o loop ./super_out/vendor.img /tmp/vendor

# Copy binaries
cp /tmp/vendor/bin/mcDriverDaemon \
   recovery/root/vendor/bin/

cp -r /tmp/vendor/bin/hw/ \
   recovery/root/vendor/bin/

# Copy libraries
cp -r /tmp/vendor/lib64/ \
   recovery/root/vendor/

# Copy firmware
cp -r /tmp/vendor/firmware/ \
   recovery/root/vendor/

# Copy TEE registry
cp -r /tmp/vendor/app/mcRegistry/ \
   recovery/root/vendor/app/

# Copy TEE thh
cp -r /tmp/vendor/thh/ \
   recovery/root/vendor/

# Step 4 — Mount system partition and copy mtk_plpath_utils
mkdir -p /tmp/system
mount -o loop ./super_out/system.img /tmp/system
cp /tmp/system/bin/mtk_plpath_utils \
   recovery/root/system/bin/mtk_plpath_utils

# Unmount when done
umount /tmp/vendor /tmp/system
